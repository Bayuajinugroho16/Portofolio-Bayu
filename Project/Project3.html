Deskripsi

Project ini berfokus pada mata kuliah Pemrograman Berbasis Website (PWEB). Pada project ini saya belajar bagaimana membuat dan mengembangkan sebuah website sederhana menggunakan HTML, CSS, dan JavaScript.

Tujuannya adalah memahami dasar-dasar struktur halaman web, mendesain tampilan agar menarik, serta menambahkan interaktivitas melalui JavaScript. Project ini juga melatih saya untuk menyusun kode dengan rapi, memisahkan file HTML, CSS, dan JS, serta membiasakan penggunaan version control (GitHub) dan deployment (Vercel).
